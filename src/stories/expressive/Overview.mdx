# expressive vainilla

`expressive vainilla` is a vanilla CSS token system that merges:

- Swiss intrinsic structure from [`libraries/swiss_intrinsic.md`](/libraries/swiss_intrinsic.md)
- Material Expressive shape/elevation behavior from [`libraries/expressive.html`](/libraries/expressive.html)

## Core Contract

- Strict 12-column asymmetrical grid with strong negative space and flush-left hierarchy
- Pill controls and rounded cards (`24px` to `32px`) with layered colored shadows
- High-contrast palette using off-white and deep charcoal surfaces plus blue, red, and green accents
- `Google Sans` for layout typography and headings
- `Roboto Mono` for data, code, snippets, and secondary technical labels

## Token Layers

| File | Layer | Purpose |
|---|---|---|
| `src/tokens/ui.ref.css` | `ui.ref` | Raw palette, spacing scale, radii, shadow primitives |
| `src/tokens/ui.sys.css` | `ui.sys` | Semantic color roles, theme switching via `data-ui-theme` |
| `src/tokens/ui.type.css` | `ui.type` | Google Sans display/body scale, Roboto Mono data classes |
| `src/tokens/ui.motion.css` | `ui.motion` | Easing + duration contracts, expression overrides via `data-ui-expression` |
| `src/tokens/ui.comp.css` | `ui.comp` | Component primitives and grid utilities |
