# Material 3 Design System

This Storybook section documents a lightweight Material 3 design system built with vanilla CSS tokens and class-based components.

## Foundations

- `Material 3/Foundations/Color Roles`
- `Material 3/Foundations/Type Scale`
- `Material 3/Foundations/Shape, Motion and Elevation`

## Components

- `Material 3/Components/Button set`
- `Material 3/Components/Cards, chips and fields`
- `Material 3/Icons/Material Symbols`

## Toolbar Controls

- `Theme`: `System`, `Light`, `Dark`
- `Expression`: `Low`, `Medium`, `High`

## CSS Entry Points

- `src/styles/material.css`
- `src/stories/material/material-story.css`

## Goals

- Keep tokens semantic (`--md-sys-*`) and reusable.
- Keep components composable for plain HTML pages.
- Keep the visual direction aligned with Material 3 expressive principles: color, shape, type, motion, and depth.
