# Cloud, Data & AI Stacks

This wiki specifically focuses on the technologies driving the Cloud, Data, and AI sectors. Here at DevClub, we explore the cutting edge of these domains.

## Cloud Computing Platforms

### Amazon Web Services (AWS)
- **Compute**: EC2, Lambda (Serverless)
- **Containerization**: EKS (Kubernetes), ECS
- **IaC**: CloudFormation, AWS CDK

### Google Cloud Platform (GCP)
- **AI/ML**: Vertex AI, Gemini API
- **Big Data**: BigQuery, Dataflow
- **App Hosting**: Cloud Run, App Engine

### Microsoft Azure
- **Enterprise**: Azure Active Directory
- **Hybrid**: Azure Arc
- **AI**: Azure OpenAI Service

## Data Engineering

### Modern Data Stack
- **Warehousing**: Snowflake, Databricks
- **ELT/ETL**: dbt (Data Build Tool), Airflow
- **Visualization**: Looker, Tableau, PowerBI

## Artificial Intelligence

### Large Language Models (LLMs)
- **Models**: GPT-4, Gemini, Claude 3, Llama 3
- **Frameworks**: LangChain, LlamaIndex
- **Vector Databases**: Pinecone, Weaviate, ChromaDB

### Machine Learning Ops (MLOps)
- **Tracking**: MLflow, Weights & Biases
- **Serving**: TorchServe, TensorFlow Serving

## Learning Paths
1. **Cloud Certification**: Start with AWS Certified Cloud Practitioner.
2. **Data Science**: Learn Python, Pandas, and SQL.
3. **AI Engineering**: Build RAG applications using LangChain.
