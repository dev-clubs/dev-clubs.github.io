# Código y Diseño

Bienvenido al club **Código y Diseño** de GDG Querétaro. Exploramos la intersección entre el desarrollo frontend/móvil y el diseño de interfaces, creando experiencias digitales excepcionales.

## Nuestro Enfoque

Creemos que el mejor software nace cuando el código y el diseño trabajan en armonía. Nuestras sesiones cubren:

- **Flutter** — Desarrollo multiplataforma con una sola base de código
- **Angular** — Framework web empresarial de Google
- **Material Design 3** — Sistema de diseño moderno y accesible
- **Web Performance** — Optimización y Core Web Vitals

## Talleres Regulares

### Design Sprints
Sesiones intensivas de diseño donde resolvemos problemas reales en equipos multidisciplinarios.

### Code Labs
Talleres prácticos paso a paso donde construimos aplicaciones completas usando las últimas tecnologías de Google.

### UI/UX Reviews
Sesiones de feedback comunitario donde analizamos diseños y prototipos para mejorarlos colectivamente.

## Stack Tecnológico

```
Frontend:    Angular, React, Lit
Móvil:       Flutter, Kotlin, Jetpack Compose
Diseño:      Figma, Material Design 3
Performance: Lighthouse, Web Vitals
```

## Proyectos de la Comunidad

Contribuimos a proyectos open source y desarrollamos herramientas para la comunidad local:

1. **Este sitio web** — Construido con Astro y vanilla CSS
2. **App de eventos** — Flutter app para la comunidad GDG
3. **Design System** — Componentes reutilizables basados en M3
